# Genesis Mesh Messaging

## One-Line Positioning

Genesis Mesh is a portable trust layer for sovereign systems.

## Not This

- Not only an AI-agent framework
- Not a marketplace
- Not a central identity provider
- Not a cloud vendor product
- Not a reputation-score system
- Not a blockchain or token platform

## Core Language

- portable trust
- sovereign systems
- recognition and revocation
- verifiable trust state
- auditable trust state
- protocol interoperability
- independent operators
- local authority
- signed evidence

## Content Rules

- Lead with systems, trust, sovereignty, recognition, revocation, and auditability.
- Treat AI agents as one use case, not the top-level frame.
- Keep claims protocol-oriented and evidence-oriented.
- Avoid implying Genesis Mesh decides who deserves trust.
- Avoid marketplace, token, reputation-score, central-registry, blockchain, and agent-framework framing.
- Tie campaign claims back to the Genesis Mesh vision, strategy, roadmap, and go-to-market source docs.
