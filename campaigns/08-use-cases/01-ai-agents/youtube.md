**Description:**
Orchestration frameworks know how to route a task between agents. They don't know whether the acting agent holds legitimate authority, under whose key it was enrolled, or whether that enrollment has been revoked — and "the framework tracks this internally" is implicit trust inside a runtime boundary, not portable trust across organizational ones.

When a user's agent under one sovereign's authority calls a specialized service run by another sovereign with no prior relationship, authorization requires two signed artifacts: an attestation of the agent's identity and capabilities, and a recognition treaty between the two sovereigns. Genesis Mesh provides exactly that layer — enrollment, attestation, recognition, revocation, evidence — available to any runtime that can make an HTTP request, independent of whatever agent framework sits on top.
