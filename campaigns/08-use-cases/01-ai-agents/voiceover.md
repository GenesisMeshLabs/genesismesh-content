Multi-agent systems have a trust problem that orchestration frameworks do not solve.

A framework can route work between agents. It does not prove which authority enrolled the acting agent, what it is allowed to do, or whether that enrollment has been revoked.

Genesis Mesh targets cross-boundary trust, not intra-runtime coordination.

When an agent from Sovereign A calls a service run by Sovereign B, the trust path should be explicit: a signed attestation, a recognition treaty, and evidence of the authorization decision.

Genesis Mesh is not an agent framework. It is the trust layer agents can consume.
