**Description:**
A branch office gateway on an unreliable link can't always phone home to verify trust. Mutual TLS proves a certificate is valid, not that the node hasn't been compromised since enrollment. Certificate authorities need revocation infrastructure that edge nodes can't reliably reach. Rotating keys across thousands of edge devices is operationally painful and frequently skipped.

Genesis Mesh gives each node a cryptographic identity it can attest offline — software version, active capabilities, current configuration — verifiable by any downstream system holding the Network Authority's public key, no live connection required. Revocation follows the same pattern: a signed artifact that establishes when trust ended and under whose key, enforced as soon as a system reconnects and checks. Identity that travels with the device, not just the connection.
