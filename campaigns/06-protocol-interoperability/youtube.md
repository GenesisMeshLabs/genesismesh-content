**Description:**
A trust layer that only works from one language or one vendor's stack isn't a trust layer — it's a library with a marketing story. Real interoperability means a Go service, a .NET tool, and a TypeScript app can each produce and verify the exact same signed artifact, byte-for-byte, against the same public key — not just compatible JSON over compatible endpoints.

Genesis Mesh defines interoperability at the cryptographic layer: a fixed signing algorithm (Ed25519), canonical JSON serialization, and a defined header set, validated by a shared conformance suite across TypeScript, Go, and .NET SDKs. That's what makes a signed attestation verifiable by a compliance team six months later, in a different codebase, with no access to the system that issued it.
