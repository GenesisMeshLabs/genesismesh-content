**Description:**
A protocol that makes developers hand-roll Ed25519 signing and canonical JSON serialization before their first trust decision won't get adopted — get one byte of the canonical body wrong and the signature fails silently, with no clue where the encoding diverged.

The Genesis Mesh Trust API exposes every operation — agreements, attestations, boundary decisions, consensus state, data-usage disclosures, trust evidence — as a stable, versioned HTTP surface. SDKs in TypeScript, Go, and .NET each provide seven sub-clients and handle request signing automatically: pass a key once, and every admin request is correctly signed from then on. The protocol layer is the hard part. The SDKs make sure the implementation isn't.
